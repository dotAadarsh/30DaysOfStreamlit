# 30DaysOfStreamlit | Day-8
## st.slider
[st.slider](https://docs.streamlit.io/library/api-reference/widgets/st.slider) allows the display of a slider input widget.
> st.slider(label, min_value=None, max_value=None, value=None, step=None, format=None, key=None, help=None, on_change=None, args=None, kwargs=None, *, disabled=False)
## st.select_slider
[st.select_slider](https://docs.streamlit.io/library/api-reference/widgets/st.select_slider) display a slider widget to select items from a list and also allows you to render a range slider by passing a two-element tuple or list as the value.
> st.select_slider(label, options=[], value=None, format_func=special_internal_function, key=None, help=None, on_change=None, args=None, kwargs=None, *, disabled=False)
